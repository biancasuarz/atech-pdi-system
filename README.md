# Sistema PDI - Plano de Desenvolvimento Individual

Sistema web para gerenciamento de Planos de Desenvolvimento Individual (PDI), permitindo o acompanhamento de metas, atividades, feedbacks e autoavaliações.

## Funcionalidades

- **Dashboard**: Visão geral com estatísticas e gráficos de evolução
- **Gestão de Metas**: Criação e acompanhamento de metas com prazos e status
- **Atividades**: Registro de atividades de desenvolvimento
- **Feedbacks**: Sistema de coleta e visualização de feedbacks
- **Notas**: Área para anotações pessoais
- **Autoavaliação**: Ferramenta de autoavaliação com diferentes critérios
- **Relatórios**: Geração de relatórios em PDF

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js (para gráficos)
- jsPDF (para geração de relatórios)
- LocalStorage (para persistência de dados)

## Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. O sistema funcionará localmente, salvando dados no LocalStorage do navegador
3. Navegue pelas diferentes seções usando o menu superior

Não há necessidade de instalação. Basta clonar ou baixar os arquivos e abrir `index.html` em um navegador moderno.
